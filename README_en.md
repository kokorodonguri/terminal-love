# ❤️ Terminal Love (`terminal-love`)

A new kind of system monitor that beats in your terminal, visualizing the "life force" of your PC.
It represents your CPU temperature and Memory usage as a stylish, beating heart ASCII art.

## ✨ Features

- **🌡️ CPU Temperature Sync**: The color of the heart changes seamlessly according to your CPU temperature (Low: Green -> Medium: Yellow -> High: Red).
- **🧠 Memory Gauge**: The "liquid" inside the heart fills up from the bottom based on your current RAM usage percentage.
- **🌊 Interactive Wave Effect**: Move your mouse cursor over the terminal to dynamically distort the space and create real-time ripples around the heart.
- **📊 Live Status Bar**: Displays real-time metrics (% and °C) in the top-left corner, along with a moving indicator (▲) on a color gradient bar.
